# Front Sight

基于 uni-app 的跨平台应用项目，使用 Vue3 + TypeScript 构建。

## 技术栈

- **框架**: uni-app
- **语言**: Vue 3 + TypeScript
- **构建工具**: Vite

## 支持平台

- H5
- 微信小程序
- 支付宝小程序
- 百度小程序
- 抖音小程序
- QQ小程序
- 快手小程序
- 京东小程序
- 飞书小程序
- 小红书小程序
- 快应用
- HarmonyOS
- App

## 开始使用

### 安装依赖

```bash
npm install
# 或
pnpm install
# 或
yarn install
```

### 开发

```bash
# H5
npm run dev:h5

# 微信小程序
npm run dev:mp-weixin

# 支付宝小程序
npm run dev:mp-alipay

# 其他平台请查看 package.json 中的 scripts
```

### 构建

```bash
# H5 构建
npm run build:h5

# 微信小程序构建
npm run build:mp-weixin

# 其他平台请查看 package.json 中的 scripts
```

## 项目结构

```
front-sight/
├── src/                 # 源代码目录
├── public/              # 静态资源目录
├── package.json         # 项目配置
├── tsconfig.json        # TypeScript 配置
└── vite.config.ts       # Vite 配置
```
